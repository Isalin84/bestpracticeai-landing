### 8. COOKIES (ФАЙЛЫ COOKIE)

Наш сайт использует файлы cookie — небольшие текстовые файлы, сохраняемые в вашем браузере.

**Типы используемых cookies:**

| Тип | Назначение | Срок хранения |
|---|---|---|
| Технические (необходимые) | Работа сайта, сессия, cookie-согласие | До закрытия браузера / 1 год |
| Аналитические (Яндекс.Метрика) | Статистика посещений | До 1 года (только с согласия) |

**Управление cookies:**  
Вы можете отказаться от cookies или удалить их в настройках вашего браузера. Обратите внимание, что отключение технических cookies может повлиять на работу сайта.

При первом посещении сайта отображается баннер с запросом вашего согласия на использование аналитических cookies. Яндекс.Метрика активируется только после вашего явного согласия.

### Что такое cookie

Cookie — это небольшие текстовые файлы, которые веб-сайты сохраняют в браузере пользователя. Они помогают сайту «запомнить» ваши действия и предпочтения.

### Какие cookie мы используем

**Строго необходимые cookie**  
Без них сайт не может корректно работать. Их использование не требует вашего согласия.
- `bp_cookie_consent` — фиксирует ваш выбор по аналитическим cookie (1 год)

**Аналитические cookie (Яндекс.Метрика)**  
Используются для понимания того, как посетители используют сайт. Активируются только после вашего согласия.
- Анонимизируют IP-адрес при возможности
- Не используются для идентификации личности

### Как управлять cookie

Вы можете изменить настройки cookie в своём браузере:
- Chrome: Настройки → Конфиденциальность → Cookie
- Firefox: Настройки → Приватность → Cookie
- Safari: Настройки → Конфиденциальность → Cookie

Также вы можете отозвать согласие на аналитические cookie, нажав кнопку «Управление cookie» в подвале сайта (в разработке) или написав нам на salinivan@mail.ru.

## ИМПЛЕМЕНТАЦИЯ В КОД

### Компонент CookieBanner

```tsx
// При первом посещении показывается баннер
// После принятия: localStorage.setItem('bp_cookie_consent', 'true')
// Яндекс.Метрика инициализируется ТОЛЬКО после принятия

const CookieBanner = () => {
  const [visible, setVisible] = useState(
    !localStorage.getItem('bp_cookie_consent')
  );
  
  const accept = () => {
    localStorage.setItem('bp_cookie_consent', 'true');
    setVisible(false);
    initYandexMetrika(); // загружаем счётчик
  };

  if (!visible) return null;
  
  return (
    <div className="cookie-banner">
      <p>
        Мы используем файлы cookie для улучшения работы сайта.{' '}
        <a href="/privacy">Политика cookie</a>
      </p>
      <button onClick={accept}>Принять</button>
      <a href="/privacy">Подробнее</a>
    </div>
  );
};
```
